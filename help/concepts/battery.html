2. Batteries¶

Batteries are objects that can be used to load specific objects from a given package. Different batteries do different things when loading the determined objects. All batteries are sub-classes of the PackageBattery class.

2.1. Reactor Battery¶

Instances of the ReactorBattery class should be used to load Reactor classes from a package. This can come in useful if you are developing an application and need a plugin system where initially unknown plugins can be loaded into the application.

2.2. Ruleset Battery¶

Instances of the RulesetBattery class should be used to load Ruleset classes from a package. This can come in useful if you are developing an application where custom rulesets can be defined.
