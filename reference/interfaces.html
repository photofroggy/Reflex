3. interfaces module¶

The interfaces module contains classes providing different interfaces for different components of the event system.

3.1. Ruleset - Binding management¶

class reflex.interfaces.Ruleset(args, mapref, output, debug)¶

Default Ruleset.

Input parameters:

  • args - Any arguments passed to the event manager object when the instance is created.
  • mapref - A reference to the dictionary storing all of the event bindings stored in the system.
  • output - A reference to the method being used to write output to the screen. This is given by the event manager object.
  • debug - Boolean defining whether or not the application is running in debug mode. This may be useful for some applications.

The Ruleset object provides an interface for managing bindings and events on a per-event basis. One Ruleset is assigned to one type of event only.

An instance of this class is used for all events that do not have a specific ruleset assigned to them.

bind(source, meth, event, options=None, *additional)¶

Create an event binding.

This is the method called by the event manager object’s own bind() method. It ensures that there are no clashes between the given binding details and any existing event bindings.

If there is a clash, then the method returns None, otherwise an instance of the Binding class is returned. The instance is also added to the object referenced by self.mapref.

init(*args)¶

init method.

This method is called by the __init__() method. Subclasses should overwrite this method if they need to do anything on initialisation.

This is a do-nothing method by default.

run(binding, data, rules, *args)¶

Run a given event binding.

Input parameters:

  • binding - An instance of the Binding class.
  • data - An instance of the Event class. This represents the event being triggered.
  • rules - This is a list of the items related to the event. Typically, these items are compared to the binding’s own options list. Custom Rulesets may use both lists in their own way, if appropriate.
  • args - Additional arguments given to the event manager’s trigger() method.
set_map(mapref)¶

Set the reference for the event map.

Input parameters:

  • mapref - As in the __init__() method, this parameter is a reference to the dictionary storing all of the event bindings being stored in the event system.

This method should only ever be called by the event manager object’s clear_bindings() method.

unbind(source, meth, event, options=None)¶

Remove an event binding.

Similar to the bind() method of this class, except the binding is removed rather than added. This method True if a matching event binding was found and removed, False if no bindings were removed.
