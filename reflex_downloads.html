This page lists links to downloads for all of the released builds for Reflex.

The name of each download file gives you some information about the build. This syntax is as follows:

  • Reflex [version] [version state] - Build [build number] ([date]-[time(24hr)]) [Build series name]

The date and time refers to when the build was released. Grab whichever version is relevent to you, but the latest download is always advised.
